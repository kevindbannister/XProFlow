# XProFlow v2 UI

This document describes the new XProFlow v2 interface styling and layout decisions.

## Layout
- App shell with a left sidebar, topbar header, and main content area.
- Rounded, glass-style cards with consistent spacing, shadows, and border radius.
- Light blue gradient background with subtle wave and dot overlays.

## Components
- `GlassCard` is the primary surface component for all panels.
- `PageHeader` and `SectionTitle` enforce consistent heading styles.
- Dashboard uses `StatCard`, `MetricTile`, and the `DonutChartCard` Recharts wrapper.

## Visual Language
- Soft blue palette, minimal borders, and calm gradients.
- Typography prioritizes readability with subtle emphasis.
- Buttons and inputs use rounded-pill styling to match the premium look.
